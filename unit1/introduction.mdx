# 에이전트 입문

<img src="https://huggingface.co/datasets/agents-course/course-images/resolve/main/en/unit1/thumbnail.jpg" alt="Thumbnail"/>

첫 번째 유닛에 오신 것을 환영합니다. 여기서는 **AI 에이전트의 기초를 탄탄하게 다질 것**입니다. 다음과 같은 내용을 다룹니다:

- **에이전트의 이해**
  - 에이전트란 무엇이며, 어떻게 작동하나요?
  - 에이전트는 어떻게 추론과 계획을 통해 의사결정을 하나요?

- **에이전트에서 LLM(대규모 언어 모델)의 역할**
  - LLM이 어떻게 에이전트의 "두뇌" 역할을 하는지
  - LLM이 메시지 시스템을 통해 어떻게 대화를 구조화하는지

- **도구와 행동**
  - 에이전트가 어떻게 외부 도구를 사용하여 환경과 상호작용하는지
  - 에이전트를 위한 도구를 어떻게 만들고 통합하는지

- **에이전트 워크플로우:**
  - *생각* → *행동* → *관찰*

이러한 주제들을 살펴본 후, `smolagents`를 사용하여 **첫 번째 에이전트를 만들어볼 것**입니다!

알프레드(Alfred)라는 이름의 에이전트가 간단한 작업을 처리하면서 이러한 개념들을 실제로 어떻게 적용하는지 보여줄 것입니다.

또한 **Hugging Face Spaces에 에이전트를 게시**하는 방법도 배워서 친구들과 동료들에게 공유할 수 있게 됩니다.

마지막으로 이 유닛의 끝에서는 퀴즈를 풀게 됩니다. 통과하면 **첫 번째 과정 인증서**인 🎓 에이전트 기초 인증서를 획득하게 됩니다.
